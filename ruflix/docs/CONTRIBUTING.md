# Contributing

Thank you for your interest in RuFlix!

The goal of this project is to build the premium cinematic frontend for Jellyfin.

## Before submitting code

Please:

- Read ARCHITECTURE.md
- Read DESIGN.md
- Follow CODE_STYLE.md

## Pull Requests

Keep pull requests focused.

One feature or one refactor per PR is preferred.

## Issues

When reporting a bug, include:

- Platform
- Browser
- Jellyfin version
- Steps to reproduce

Screenshots are always appreciated.

## Feature Requests

Before requesting a feature, ask:

Does this improve the viewing experience?

If the answer is yes, we'd love to discuss it.
