# RuFlix Design Philosophy

RuFlix is not intended to be another Jellyfin client.

It is intended to be the premium cinematic experience for personal media.

## Principles

### Content First

The media should always be the hero.

Artwork, logos, and ambient lighting should carry the experience.

The interface should stay out of the way.

---

### TV First

RuFlix is designed for the living room first.

Everything should be usable from:

- Mouse
- Keyboard
- Remote
- Game Controller

Hover states should always have a keyboard/controller equivalent.

---

### Cinematic

The interface should feel alive.

Examples include:

- Ambient backdrop lighting
- Smooth transitions
- Crossfading hero artwork
- Elegant motion
- Layered depth

Motion should enhance the experience, never distract from it.

---

### Comfortable

Users often browse for several minutes before choosing something.

The UI should reduce fatigue through:

- Spacious layouts
- Clear typography
- Consistent spacing
- Minimal clutter

---

### Personal

RuFlix should feel like *your* streaming service.

Examples:

- Friday Night shelves
- Holiday recommendations
- Family Movie Night
- Seasonal collections

RuFlix should create experiences, not simply display libraries.

---

## Design Language

RuFlix favors:

- Large artwork
- Official logos
- Glass surfaces
- Soft gradients
- Deep blacks
- Warm lighting
- Smooth animation

---

## Non-Goals

RuFlix is **not** trying to imitate Netflix or Plex.

Instead, it should establish its own recognizable identity while embracing modern streaming design principles.

A screenshot of RuFlix should be recognizable without seeing the logo.
