# 🍿 RuFlix

> The premium cinematic frontend for Jellyfin.

RuFlix is an open-source media frontend focused on immersion, atmosphere, and beautiful design.

Instead of simply displaying a media library, RuFlix is designed to make browsing your personal collection feel like using a modern streaming platform.

---

## Vision

RuFlix is built around a few simple ideas.

- 🎬 Content comes first.
- 🍿 The interface should disappear.
- 🌌 Browsing should feel cinematic.
- 🛋️ Designed for the living room.
- ❤️ Your media deserves a premium experience.

## Philosophy

RuFlix is not trying to clone Netflix or Plex.

Instead, it aims to create its own identity while embracing modern streaming UX principles.

The goal is simple:

> Your Jellyfin server should feel like your own streaming service.
