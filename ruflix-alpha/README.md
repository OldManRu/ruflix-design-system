# RuFlix Alpha

RuFlix Alpha is the production-track React + Vite + TypeScript application for the RuFlix media client.

The current HTML prototype remains in `prototypes/ruflix-home/` as the frozen design reference while this app becomes the long-term client foundation.

## Alpha goal

The first alpha is successful when someone can open RuFlix, connect to Jellyfin, browse Home, open Media Details, and start or resume playback from the couch.
