# Sprint 5: Media Details

## Objective

Begin the Alpha track and design the first real detail experience.

## Scope

- Keep the existing Home prototype as the reference implementation.
- Move long-term development into `ruflix-alpha/`.
- Build toward a React, Vite, and TypeScript application.
- Establish reusable building blocks for Home, Media Details, and Playback.

## Media Details goals

- Open a selected title from Home.
- Present backdrop, logo, metadata, description, and primary actions.
- Prioritize Resume when progress exists.
- Return to Home with focus preserved.
- Keep all interactions TV-first.

## Alpha milestone

RuFlix becomes Alpha when Home, Details, Playback, basic settings, and Jellyfin playback are usable from the couch.
