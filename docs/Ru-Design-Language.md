# Ru Design Language (RDL)

Version: 0.1 (Living Document)

## Philosophy
- Modern
- Calm
- Premium
- Alive
- Comfortable
- Family-friendly
- Never 'developer-looking'

Design principle: Calm Confidence.

## Current Scope
Applies to:
- RuDash
- RuPics
- Future media application
- RuID (future)

## Status
- Typography: Stable
- Primary Color: Experimental
- Cards: Stable direction
- Motion: Stable direction
- Shadows: TBD
- Glass Effects: TBD
- Icon Style: TBD
- Logo Family: TBD

## Typography
- Font: Inter (initial recommendation)
- Goal: Neutral, readable, modern.

## Color
Target direction:
- Dusty / steel blue primary.
- Dark neutral backgrounds.
- Comfortable contrast.

## Cards
Follow the rounded, spacious style established in the current media app prototypes.

## Motion
- Subtle
- Rich
- Purposeful
- No distracting animation
- Microinteractions should reward observation.

## RuPics
- Aperture-based logo.
- Aperture may subtly move at randomized intervals (~1–5 minutes).
- Broad redesign rather than incremental sprints.

## Design Principles
- Calm Confidence
- Motion Has Purpose
- Human First
- Content Is the Hero
- Comfortable Density
- Build for Longevity

## Notes
This is a living document. Confirmed facts should remain distinct from future ideas and proposals. Experimental items should be clearly marked until adopted.
