# Product Direction

RuFlix is a personal media viewing experience, not a broad developer platform.

The design system exists to support a comfortable, polished, and approachable environment for browsing, organizing, and watching personal media collections.

## Primary Objective

Create a calm, cinematic interface that makes personal media feel easy to browse, enjoyable to view, and comfortable to use from a couch, desktop, tablet, or shared family screen.

## What RuFlix Should Be

- Comfortable for personal and family media libraries
- Cinematic without being overwhelming
- Simple enough for everyday use
- Accessible and readable in low-light environments
- Friendly to remote control, keyboard, mouse, and touch input
- Flexible enough to support movies, shows, home videos, photos, and collections

## What RuFlix Should Not Be

- A generic enterprise design system
- A full developer platform
- A large UI framework competing with general-purpose component libraries
- A clone of Netflix, Plex, Jellyfin, or any other existing product
- A feature-heavy dashboard that distracts from watching media

## Experience Principles

### 1. Media First

The content should be the star. UI chrome should support browsing and playback without competing with posters, thumbnails, artwork, or video.

### 2. Comfortable Viewing

RuFlix should feel natural in relaxed environments. Spacing, typography, contrast, and motion should be tuned for longer sessions and lean-back viewing.

### 3. Calm Navigation

Movement through the interface should be predictable. Users should always understand where they are, what is selected, and how to get back.

### 4. Personal, Not Corporate

The tone should feel warm and familiar. This is a home media experience, not a business application.

### 5. Accessible by Default

The system should support readable text, visible focus states, keyboard navigation, sufficient contrast, screen readers, and reduced motion preferences.

## Core Screens to Support

- Home
- Library
- Media details
- Continue watching
- Collections
- Search
- Playback controls
- Settings
- Profiles or household members

## Design Mood

RuFlix should feel:

- cinematic
- quiet
- premium
- personal
- warm
- low-friction
- modern

The visual direction can use depth, glow, blur, and dark surfaces, but these choices should never make the product harder to read or use.
