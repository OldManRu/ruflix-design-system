# Typography

RuFlix typography should be readable from both desktop distance and couch distance.

## Goals

- Prioritize readability over decoration
- Support scanning through media titles and metadata
- Make selected content feel cinematic without sacrificing clarity
- Avoid tiny text in primary flows

## Type Roles

| Role | Usage |
| --- | --- |
| Display | Hero titles and featured media |
| Heading | Section titles and page titles |
| Title | Media cards, dialogs, and detail panels |
| Body | Descriptions and supporting copy |
| Label | Buttons, tabs, controls, and metadata |
| Caption | Secondary details such as runtime, year, rating, and season info |

## Guidance

- Prefer medium and semibold weights for selected titles and actions.
- Avoid long uppercase labels except for small metadata chips.
- Use generous line-height for descriptions.
- Ensure text remains readable over artwork by using scrims, gradients, or solid surfaces.

## Viewing Distance

Personal media interfaces may be used across laptops, TVs, tablets, and mounted displays. Typography should scale comfortably and avoid relying on dense text layouts.
