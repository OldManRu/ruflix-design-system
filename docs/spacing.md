# Spacing

RuFlix spacing should make the interface feel calm, breathable, and easy to navigate.

## Goals

- Give media artwork enough room to stand out
- Prevent controls from feeling cramped
- Support remote, keyboard, touch, and mouse interaction
- Create predictable rhythm across shelves, cards, dialogs, and playback controls

## Spacing Scale

RuFlix uses a simple 4px-based spacing scale.

| Token | Value | Usage |
| --- | ---: | --- |
| `space-1` | 4px | Tight internal spacing |
| `space-2` | 8px | Compact gaps |
| `space-3` | 12px | Small component padding |
| `space-4` | 16px | Default component padding |
| `space-5` | 20px | Comfortable control groups |
| `space-6` | 24px | Card and panel spacing |
| `space-8` | 32px | Section spacing |
| `space-10` | 40px | Large layout separation |
| `space-12` | 48px | Hero and page-level spacing |
| `space-16` | 64px | Major screen regions |

## Guidance

- Use more space around primary media surfaces.
- Keep metadata compact but readable.
- Avoid dense dashboard-like layouts.
- Make selected and focused elements feel intentional, not crowded.
