# Home Screen

The Home screen is the heart of RuFlix. It should answer one question immediately: what do I want to watch next?

## Purpose

Home should make the library feel personal, current, and easy to enter.

## Primary Regions

1. Adaptive left navigation
2. Search
3. Featured hero media
4. Continue Watching
5. Recently Added
6. Family Favorites
7. Conditional shelves based on available content

## Navigation

The left navigation should change based on actual available content. Do not show Photos, Music, Live TV, Collections, or other optional areas unless the related content or feature exists.

## Hero Behavior

The hero should feel cinematic but calm.

- Use large artwork or backdrop imagery.
- Apply subtle ambient background color derived from the artwork.
- Keep actions simple: Play / Resume, More Info, optional Trailer.
- Avoid autoplay trailers by default.

## Shelf Priority

1. Continue Watching
2. Recently Added
3. Family Favorites
4. Movies
5. TV Shows
6. Live TV if configured
7. Optional content shelves if available

## Movie Night Entry

Movie Night may appear as a lightweight shelf or action. It should surface a few watchable options and a Surprise Me action without becoming a separate complex mode.

## Motion

- Gentle focus scale
- Soft elevation on selected cards
- Smooth shelf movement
- Slow ambient background transitions
- No attention-grabbing animation

## Success Criteria

A user should be able to open RuFlix, understand what is available, and begin watching something without thinking about the interface.
