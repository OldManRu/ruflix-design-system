# Screen Inventory

This inventory defines the major RuFlix screens and their purpose.

## Core Screens

### Home

The starting point for watching media. Prioritizes Continue Watching, Recently Added, Family Favorites, and available content types.

### Media Details

The decision screen. Gives enough information to confidently press Play without overwhelming the user.

### Playback

The quietest screen in the product. Playback controls should appear only when needed and disappear quickly.

### Library

A focused browsing space for Movies, TV Shows, Live TV, and available optional media types.

### Search

A fast way to find known content. Search should be forgiving and visually simple.

### Settings

Configuration without clutter. Settings should be clear, grouped, and not allowed to dominate the viewing experience.

## Secondary Screens / Experiences

### Movie Night

A lightweight Home screen action or shelf that helps reduce decision fatigue. It should remain optional and simple.

### Collections

Curated or automatic groupings such as franchises, holiday collections, family favorites, or personal sets.

### Profiles

Optional household profiles for personalization, visibility, and watch progress.

### Live TV Guide

A focused guide for available live TV content when configured.

### Photos

Only visible when photo libraries exist.

### Music

Only visible when music libraries exist.

## Screen Prioritization

Sprint order:

1. Home
2. Media Details
3. Playback
4. Search
5. Library
6. Settings
7. Movie Night
8. Collections
9. Profiles
10. Optional media types

## Rule

No screen should exist only because other media apps have it. Every screen must serve the comfort of browsing, choosing, or watching personal media.
