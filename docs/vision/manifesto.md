# The RuFlix Manifesto

RuFlix was not created to organize files.

It was created to make spending time with your personal media feel effortless, beautiful, and comfortable.

Every movie has a memory.
Every television series tells a story.
Every home video captures a moment that cannot be recreated.

Those moments deserve an interface that respects them.

The interface should disappear.
The atmosphere should remain.

We believe media should be the hero.
We believe comfort is a feature.
We believe motion should create atmosphere, not distraction.
We believe beautiful software earns trust through restraint.

RuFlix is not trying to be the application with the most features.
It is trying to be the application people enjoy spending time in.

Every pixel earns its place.
Every animation has intention.
Every interaction should help someone relax.

When in doubt, choose simplicity.
When given the choice, choose comfort.

Welcome home.
